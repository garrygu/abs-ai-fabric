Usage

To add this header to any application:

<!-- Add to your HTML -->
<script type="module" src="http://localhost:5173/abs-header.js"></script>

<abs-unified-header 
    app-id="your-app-id"
    app-name="Your App Name"
    app-icon="üì±"
    hub-url="http://localhost:5173"
    gateway-url="http://localhost:8081">
</abs-unified-header>

Configuration

The component accepts the following attributes:

  • app-id - Unique identifier for the current app
  • app-name - Display name shown in the header
  • app-icon - Emoji or text icon for the app
  • hub-url - URL of the AI Fabric Hub (for "Return to Hub")
  • gateway-url - URL of the gateway API (for fetching apps list)
